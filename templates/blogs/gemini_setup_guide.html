Complete Guide: Get Google Gemini API & Configure in Lexis Tools Dashboard

Learn how to obtain your Google Gemini API key and seamlessly integrate it into your Lexis Tools dashboard for enhanced translation and voice capabilities.

Dec 15, 2025 • 10 min read

Google Gemini API provides powerful AI capabilities for translation and voice synthesis. This comprehensive guide will show you exactly how to get your API key and configure it in your Lexis Tools dashboard settings.

Part 1: Getting Your Google Gemini API Key

Step 1: Access Google AI Studio

First, navigate to Google AI Studio and sign in with your Google account.

Step 2: Navigate to API Keys

Once logged in, look for the "Get API Key" button or navigate to the API Keys section in the left sidebar.

Part 2: Adding API Key to Your Lexis Tools Dashboard Settings.

Step 1: Access Your Lexis Tools Dashboard Settings.

Log into your Lexis Tools account and navigate to your dashboard settings. You can access it directly using this link: Lexis Tools Dashboard

Lexis Tools Dashboard Settings

In the Settings page, you'll find two main configuration sections:

LLM Configuration

Use this section to configure Google Gemini for text translation and generation tasks.

Voice Configuration

Use this section to configure Google Gemini for text-to-speech and voice synthesis.

Step 2.1: Configure LLM Settings

Follow these steps to add your Google Gemini API key:

For LLM Configuration:

  1. Select "Google" as your LLM Provider
  2. Choose your preferred model (e.g., "gemini-2.5-pro")
  3. Paste your API key in the "API Key" field
  4. Click "Save Configuration"

LLM Configuration Section - Settings Form

Step 2.2: Voice Configuration Settings

Follow these steps to add your Google Gemini API key:

For Voice Configuration:

  1. Select "Google" as your TTS Provider
  2. Choose your preferred voice model
  3. Select a voice name from the dropdown
  4. Paste the same API key in the "API Key" field
  5. Click "Save Configuration"

API Key Input Fields - Configuration Form

Troubleshooting Common Issues

API Key Not Working

Double-check that you copied the entire API key without any extra spaces. Ensure your Google Cloud project has billing enabled if required.

Model Not Available

Some models may not be available in all regions. Try using "gemini-2.5-flash" as it has the widest availability.

Configuration Not Saving

Make sure you're logged into your Lexis Tools account and have the necessary permissions. Try refreshing the page and attempting again.
