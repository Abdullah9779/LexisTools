Complete Guide: Get Groq Cloud API & Configure in Lexis Dashboard

Learn how to obtain your Groq Cloud API key and integrate it into your Lexis dashboard for lightning-fast AI processing and translation capabilities.

Dec 11, 2025 • 8 min read

Groq Cloud offers ultra-fast inference speeds for large language models, making it perfect for real-time translation and AI applications. This guide will walk you through getting your API key and configuring it in your Lexis dashboard.

Part 1: Getting Your Groq Cloud API Key

Step 1: Access Groq Console

Navigate to Groq Console and create an account or sign in with your existing credentials.

Groq Console Homepage - Main Dashboard

Step 2: Navigate to API Keys

Once logged in, look for the "API Keys" button or navigate to the API management area.

API Keys Section - Navigation Interface

Step 3: Create New API Key

1. Click on "Create API Key" or "New API Key" button.

Groq API Key Section.

2. Give your API key a descriptive name (e.g., "Lexis Tools Api") and click "Submit" button to generate the API key.

Groq API Key Dilogbox.

3. Click on "Copy" button to copy api key to clipboard.

Groq API Key.

Security Warning: Store your Groq API key securely! Never share it publicly or commit it to version control. Treat it like a password.

Part 2: Adding Groq API Key to Your Lexis Tools Dashboard Settings

Step 1: Access Your Lexis Tools Dashboard Settings.

Log into your Lexis Tools account and navigate to your dashboard settings. You can access it directly using this link: Lexis Dashboard Settings

Follow these steps to configure Groq in your Lexis dashboard:

LLM Configuration Steps:

  1. Select "Groq" as your LLM Provider from the dropdown
  2. Choose your preferred model from the available options
  3. Paste your Groq API key in the "API Key" field

Lexis Tools Dashboard Settings

Step 2: Save Your Configuration

Click "Save Configuration" to apply the configuration.

Dashboard Settings Navigation - Settings Menu

Available Groq Models in Lexis

Supported Models:

  • • openai/gpt-oss-120b - High-performance general model
  • • groq/compound - Optimized compound model
  • • meta-llama/llama-4-maverick-17b-128e-instruct - Latest Llama variant
  • • moonshotai/kimi-k2-instruct-0905 - Specialized instruction model
  • • qwen/qwen3-32b - Qwen series model

Tip: Start with groq/compound for the best balance of speed and quality, or use meta-llama/llama-4-maverick-17b-128e-instruct for more complex translation tasks.

Troubleshooting Common Issues

API Key Authentication Failed

Verify that you copied the complete API key without any extra spaces. Check that your Groq account has sufficient credits and the API key hasn't expired.

Model Not Available

Some models may have limited availability or require special access. Try using "groq/compound" as it has the most reliable availability.

Rate Limiting

If you encounter rate limits, check your Groq console for usage limits and consider upgrading your plan for higher throughput.
