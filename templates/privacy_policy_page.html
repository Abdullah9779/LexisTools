Privacy Policy

Last updated: {% now "F d, Y" %}

This Privacy Policy explains how we collect, use, and protect your information when you use our website.

1. Information We Collect

We collect the following information when you create an account or use our services:

1.1 Account Information

  • First Name
  • Last Name
  • Username
  • Email Address
  • Password (stored securely using hashing)

1.2 User Settings & Configurations

After logging in, you may optionally provide additional data inside your account settings:

LLM Configuration:

  • API Key (provided by the user)
  • Model Name
  • Provider Name
  • Any custom LLM configuration fields you set

Voice Configuration:

  • Provider Name
  • Model Name
  • Voice Name
  • API Key (optional, provided by the user)

Prompts:

  • Prompt Name
  • Prompt Text

You choose whether to provide these details. You may also revoke or delete your API keys from your provider at any time.

2. How We Use Your Information

We use your data only for the following purposes:

  • To create and manage your user account
  • To authenticate your login
  • To run LLM or Voice services using your own API keys
  • To apply the configurations you set inside your dashboard
  • To improve platform functionality and user experience

We do not use your data for advertising, marketing, or tracking.

3. No Third-Party Data Sharing

We do not share, sell, or transfer your data to any third-party organizations.

The only time your API key is used is when:

  • You configure a provider, and
  • The system sends requests directly to that provider using your key, under your permission.

Your keys are never shown to any other user, and we do not access them manually.

4. Data Security

We take your security seriously. Your data is protected through:

  • Encrypted HTTPS communication
  • Secure password hashing
  • Restricted internal access
  • Safe storage practices

You are responsible for keeping your API keys safe and rotating or revoking them from your provider if needed.

5. User Control & Data Rights

You can:

  • Update your profile information
  • Change your API keys
  • Modify your LLM/Voice settings

If you delete your account, we permanently remove your stored data.

6. Cookies & Tracking

We only use essential cookies for:

  • Login sessions
  • Security

We do not use tracking cookies, analytics trackers, or advertising cookies.

7. Changes to This Privacy Policy

We may update this Privacy Policy as the platform evolves. If we make significant changes, we will notify users on the website.

8. Contact Us

If you have questions regarding this Privacy Policy, you can contact us at: support@lexistools.com
